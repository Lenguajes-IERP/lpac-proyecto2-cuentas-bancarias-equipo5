# Proyecto compartido

Este directorio contiene las clases compartidas entre backend y WPF.

## Contenido

```text
SalesPro.Domain      Entidades y excepciones del dominio
SalesPro.Contracts   DTOs, requests y responses usados por API y WPF
```

## Regla

Los cambios en esta carpeta deben revisarse con cuidado porque pueden afectar tanto al backend como al frontend WPF.
